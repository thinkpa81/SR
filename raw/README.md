# Raw KLIPS Files

This directory is a placeholder for local KLIPS public-use raw files.

The raw files are not redistributed in this GitHub package. Download the public-use KLIPS files, codebooks, questionnaires, and user guide from the official Korea Labor Institute KLIPS portal under the applicable data-use terms, then place the required Excel files here before running the full pipeline.

See `config/expected_raw_files.csv` for the expected local Excel filenames and hashes used for the submitted analysis.
